Description:
During the construction of the portfolio, I could create mostly everything according to the original plans. I thought it would be a good idea to be able to change the language on the website, perhaps the implementation of this was the most thought-provoking. There was one thing that I could not solve dough, if we move the cursor to a block in the navigation (menu), some parameter of the block will change (with the help of hover). But if we click on it and open that "menu tab", that change disappears from the block. I wanted this "change" to reamin active on the actual block, so if we select a block in the menu, it will remain selected until we move to another menu, even if the cursor isn't on it at the moment (so we can see on which block we are).
